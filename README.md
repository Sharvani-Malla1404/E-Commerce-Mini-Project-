## Project Overview

This repository hosts a fully deployed, interactive Web-Based E-Commerce Application. 
Originally architected as a core Java retail system, the project was scaled into a full-stack application using Replit AI to implement a responsive web user interface (UI).
The application delivers a seamless retail experience, allowing users to browse a dynamic product catalog, manage a live shopping cart, and view automated checkout bills in real time.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Core Features

* **Interactive Web UI:** Replaces terminal interactions with a clean, modern, and user-friendly web interface for a smooth shopping experience.
* **Dynamic Product Catalog:** Displays real-time item availability, pricing structures, and current stock counts.
* **Live Cart Management:** Allows users to add or remove specific quantities of products with instantaneous visual feedback.
* **Real-Time Inventory Synchronization:** Automatically updates and reflects global stock availability as items move in and out of the user's cart.
* **Automated Billing & Checkout:** Dynamically processes active cart contents to calculate accurate, itemized financial totals upon checkout.

------------------------------------------------------------------------------------------------------------------------------------------------------

## Architecture & Technical Highlights

* **Robust Backend Logic:** Powered by Java object-oriented principles, using structured methods and tracking matrices to handle application state, inventory changes, and data calculation.
* **AI-Driven Full-Stack Integration:** Leveraged Replit AI to bridge core Java functionalities with modern front-end UI elements, transforming business logic into a production-ready web application.
* **Cloud Deployment:** Fully containerized and hosted live via Replit, making the application accessible instantly through any web browser.
